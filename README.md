# APIprodutosJava
Uma API em JAVA para um sistema de controle de produtos

#Dependências e tecnologias utilizadas:
o Spring Web 
o Spring Data JPA 
o MySQL Driver 
o Lombok
o Java ( 17 ou superior)
o VSCode

#Como inicializar
No seu VSCode clicar no botão RUN JAVA

#Porta da API
http://localhost:8080
