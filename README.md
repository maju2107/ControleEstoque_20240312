# Controle de Estoque
Uma API em JAVA para um sistema de controle de estoque de produtos

# Dependências e tecnologias utilizadas:
- Spring Web 
- Spring Data JPA 
- MySQL Driver 
- Lombok
- Java ( 17 ou superior)
- VSCode

# Como inicializar
- No seu VSCode clicar no botão RUN JAVA

# Porta da API
- http://localhost:8080
